*** linux目录
    1. /bin 存放系统命令的目录,单用户模式下可以执行 
    2. /sbin 保存和系统环境设置相关的命令,只有超级用户可以使用这些命令,
    3. /usr/bin 存放系统命令,单用户模式下不可以执行
    4. /usr/sbin 存放系统命令不必要的系统管理命令
    5. /boot 系统启动目录
    6. /dev 设备文件
    7. /etc 配置文件,账号密码,启动脚本....

**** 文件搜索
 1. 根据文件名搜索
	#+BEGIN_SRC 
	find  /etc -name filename
	-iname 不区分大小写
	#+END_SRC
     2. 根据文件大小搜索
	#+BEGIN_SRC 
	find / -size +204800
	根目录查找大于100MB的文件
	+n 大于 -n小于 n等于
	#+END_SRC
     3. 根据所有者查找
	#+BEGIN_SRC 
	find /home -user root
	#+END_SRC
     4. 根据所属组查找
	#+BEGIN_SRC 
	find /home -group root
	#+END_SRC
     5. 根据文件属性查找
	#+BEGIN_SRC 
	find /etc -cmin -5
	在/etc下查找5分钟内被修改过属性的文件和目录
	-amin 访问时间 access
	-cmin 文件属性 change
	-mmin 文件内容 modify
	#+END_SRC
     6. 根据文件类型查找
	#+BEGIN_SRC 
	find -type f 文件 d 目录 l 软件连接
	find /etc -name init* -a -type -d 
	查找以init开头的目录 -a 连接作用  -o 2选一
	#+END_SRC

**** 解压缩
     1. gzip 压缩只能压缩文件不能压缩目录
	#+BEGIN_SRC 
	gzip 文件 
	后缀 .gz
	#+END_SRC
     2. gunzip 解压缩
	#+BEGIN_SRC 
	gunzip *.zip
	#+END_SRC
     3. tar 可以压缩目录
	#+BEGIN_SRC 
	tar cvf 目标名字 目录
	打包压缩
	tar zcvf 目标 文件  z 是.gz
	tar cjf 可以生成.tar.bz2格式压缩包
	解压缩
	tar xjf 文件
	#+END_SRC
     4. zip
	#+BEGIN_SRC 
	-r 压缩目录
	zip  选项 压缩后的文件名 文件或目录
	后缀 .zip
	#+END_SRC
     5. unzip 1.zip 解压缩
     6. bzip2
	#+BEGIN_SRC 
	bzip2 选项-k 文件 
	-k 产生压缩文件后保留原文件
	后缀.bz2
	#+END_SRC
	- 压缩
	  #+BEGIN_SRC 
	  bzip2 -k bak.tar
	  #+END_SRC

	- 解压缩
	  #+BEGIN_SRC 
	  bunzip2 filname -k 
	  -k 保留压缩包
	  #+END_SRC

** 端口
   查看linux进程占用的端口号
   1. ss
      用于转储套接字统计信息
      ss -tnlp|grep ssh
      ss -tnlp|grep ":22"
      t tcp
      n 端口号
      l 监听的
      p 进程
   2. netstat
      可以显示打开的套接字列表，能够显示网络链接，路由表，接口统计信息，伪装连接及多播成员
      netstat -tnlp|grep ssh
      netstat -tnlp|grep ":22"
   3. lsof
      可以列出打开的文件
      lsof -i -P |grep ssh
      lsof - tcp:22
   4. fuser
      可以列出那些打开了文件的进程的进程id
      fuser -v 22/tcp
   5. nmap
      网络检测工具和端口扫描工具
      nmap -sV -p 22 localhost
   6. systemctl
      控制管理其的服务管理器
