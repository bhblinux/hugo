*** 函数
    - 引用传递
      1. 定义
	 public void add(ref int a);
      2. 调用
	 add(ref a);
    - 值传递
      1. 定义
	 public void add(int a,int b);
      2. 调用
	 add(a,b);
    - 可变参数
      1. 定义
	 public void printArr(params int[] arr);
      2. 调用
	 printArr(arr);

    - 按输出传递参数
      1. 定义
	 public void getVal(out int a);

      2. 调用
	 getVal(out a);

    - 重载
      函数名一样,参数个数或者参数类型不同即为重载,返回值不同,不构成重载
